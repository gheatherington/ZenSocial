**NavigationState.swift** (NEW) -- per D-06, D-07:

Create `@Observable @MainActor class NavigationState` with:

1. `enum Screen: String` with cases `home`, `instagram`, `youtube`
2. `@ObservationIgnored @AppStorage("lastPlatform") private var lastPlatformRaw: String = ""`
3. `var activeScreen: Screen = .home` -- transient, drives UI
4. `var isPillExpanded: Bool = false` -- transient
5. `var pillPosition: CGPoint` -- loaded from UserDefaults on init
6. `var activePlatform: Platform?` -- computed from activeScreen (returns nil for .home, .instagram for .instagram, .youtube for .youtube)
7. `func navigate(to screen: Screen)` -- sets activeScreen, collapses pill (`isPillExpanded = false`), saves to lastPlatformRaw if not .home
8. `func restoreLastPlatform()` -- reads lastPlatformRaw, sets activeScreen if valid non-home screen, else stays .home (per D-06: first launch = home, subsequent = last platform)
9. `func savePillPosition()` -- writes `pillPosition.x` and `pillPosition.y` to UserDefaults keys `"pillPositionX"` and `"pillPositionY"` (per D-07)
10. `func loadPillPosition()` -- reads from UserDefaults, sets pillPosition if nonzero, else uses default `CGPoint(x: UIScreen.main.bounds.width - 44, y: UIScreen.main.bounds.height - 120)` (per D-07)
11. `init()` calls `loadPillPosition()`

```
            Text("ZenSocial")
                .font(.largeTitle)  // 34pt
                .fontWeight(.bold)
                .foregroundStyle(.white)
            Text("Social media, without the noise.")
                .font(.subheadline)  // 15pt
                .foregroundStyle(.white.opacity(0.6))
        }

        // Platform launcher cards + settings
        VStack(spacing: 16) {  // md spacing between cards
            ForEach(Platform.allCases) { platform in
                PlatformLauncherCard(platform: platform) {
                    onSelectPlatform(platform)
                }
            }
            // Settings card (per D-11 -- tappable, not grayed out)
            SettingsLauncherCard {
                showSettings = true
            }
        }
        .padding(.horizontal, 16)  // md horizontal padding
    }
}
.navigationDestination(isPresented: $showSettings) {
    SettingsPlaceholderView()
}
```
