---

## Floating Button Expansion

| Option | Description | Selected |
|--------|-------------|----------|
| Radial/arc menu | Icons fan out in a circle around the button. Compact, no labels by default. | |
| Vertical popover list | Small card with labeled rows. More readable. | |
| Pill/dock | Horizontal pill expands from the circular button with icon items. | ✓ |

**User's choice:** Pill/dock — circular button expands into a horizontal pill with icon-based buttons (Home, Instagram, YouTube). No labels unless long-pressed.

**Notes:** User specifically wants Settings to remain on the home screen only, not in the pill — keeps the switcher lean and scales better when more platforms are added.

---

## Launch Behavior

| Option | Description | Selected |
|--------|-------------|----------|
| Always home screen | Consistent branded entry point every time. | |
| Last platform used | App resumes where you left off. | |
| First launch = home, subsequent = last platform | Onboarding-friendly + resumable. | ✓ |

**User's choice:** First launch → home screen. Subsequent launches → last platform used, saved via `@AppStorage`.

---

## Button Position Persistence

| Option | Description | Selected |
|--------|-------------|----------|
| Save to UserDefaults | Button reappears exactly where the user left it. | ✓ |
| Always reset to default corner | Simpler, predictable. | |
| Snap to nearest corner | Remembers last corner, not exact coordinates. | |

**User's choice:** Save full drag position to UserDefaults and restore on launch.

---

## Transition Animations

| Option | Description | Selected |
|--------|-------------|----------|
| Fade | Simple, minimal, fits dark aesthetic. | ✓ |
| Push/slide | Directional, feels like navigation depth. | |
| Zoom/scale | Very iOS 17+ native, slightly playful. | |
| No animation | Maximum speed, zero frills. | |

**User's choice:** Fade.

**Notes:** User explicitly noted animations are a revisit candidate — "we may revisit in the future as we slowly develop a better idea of how the app should look."

## Settings Stub

| Option | Description | Selected |
|--------|-------------|----------|
| Grayed-out / disabled | Visible but non-interactive. | |
| Tappable → empty screen | Navigates to a placeholder screen. | ✓ |

**User's choice:** Tappable → empty placeholder screen.
