## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | XCTest (built into Xcode) |
| **Config file** | none — XCTest requires no config file |
| **Quick run command** | `xcodebuild build -scheme ZenSocial -destination 'platform=iOS Simulator,name=iPhone 16' 2>&1 \| tail -20` |
| **Full suite command** | `xcodebuild test -scheme ZenSocial -destination 'platform=iOS Simulator,name=iPhone 16' 2>&1 \| tail -40` |
| **Estimated runtime** | ~60 seconds (build) / ~120 seconds (test) |

## Sampling Rate

- **After every task commit:** Run quick build command (build must succeed)
- **After every plan wave:** Run full suite + manual simulator walkthrough
- **Before `/gsd:verify-work`:** Full suite must be green + manual visual verification
- **Max feedback latency:** 120 seconds
